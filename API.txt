
* Get All Topic has PUBLIC status
- End point: /api/topic/public
- Method: GET
- Header: Authorization -> "Bearer ${token}"
- Body: null

* Get All Topic of Account
- End point: /api/topic/
- Method: GET
- Header: Authorization -> "Bearer ${token}"
- Body: null

* Add new Topic for Account
- End point: /api/topic/
- Method: POST
- Header: Authorization -> "Bearer ${token}"
- Body: {topicName, desc, isPublic, words}
    (words: list words, example for list words in file Ex.docs)

* Edit Topic
- End point: /api/topic/:id
- Method: PATCH
- Header: Authorization -> "Bearer ${token}"
- Body: [topicName, desc, isPublic, words]
    (words: list words, example for list words in file Ex.docs)

* Delete Topic
- End point: /api/topic/:id
- Method: DELETE
- Header: Authorization -> "Bearer ${token}"
- Body: null


* Get Topic by ID of Account or Shared Topic
- End point: /api/topic/:id
- Method: GET
- Header: Authorization -> "Bearer ${token}"
- Body: null

* Add new words to Topic
- End point: /api/topic/:id/word
- Method: POST
- Header: Authorization -> "Bearer ${token}"
- Body: {words}

* Edit words in Topic
- End point: /api/topic/:id/word/:wordid
- Method: PATCH
- Header: Authorization -> "Bearer ${token}"
- Body: {words}

* Delete words in Topic
- End point: /api/topic/:id/word/:wordid
- Method: DELETE
- Header: Authorization -> "Bearer ${token}"
- Body: null
